# react-web-os
création d'un WebOS avec un design complètement custom mais avec une inspiration du design de Windows 11 en ReactJS

# Get started

## Install
```shell
npm install
or
yarn install
or
pnpm install
```

## Start project
```shell
npm run dev
or
yarn dev
or
pnpm run dev
```

## Start StoryBook
```shell
npm run storybook
or
yarn storybook
or
pnpm run storybook
```
